# ContactsManager
Application Java de gestion de contacts réalisée en programmation orientée objet.

## Description

Application Java permettant de gérer des contacts.

## Fonctionnalités

- Ajouter un contact
- Rechercher un contact
- Afficher son numéro de téléphone

## Technologies

- Java
- Programmation Orientée Objet
